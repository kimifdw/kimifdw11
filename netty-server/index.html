      ByteBuf简介

  • 创建buffer使用Unpooled或Pooled。
  • discardable bytes。无效空间，可丢弃字节的区域
  • readable bytes。内容空间，可读字节的区域，由readerIndex和writerIndex指针控制
  • writable bytes。空闲空间，可写入字节的区域，由writerIndex指针和capacity容量控制
